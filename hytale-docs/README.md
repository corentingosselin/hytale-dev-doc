# Hytale Development Documentation

Community-driven documentation for Hytale plugin development, built from real Discord conversations where developers shared solutions and knowledge.

## About

This documentation site compiles insights, solutions, and discussions from the Hytale developer community. It represents collective knowledge from developers helping each other solve problems and discover best practices.

## Features

- **Getting Started**: Setup guides for Java, server installation, and first plugin
- **Server Setup**: Configuration, hosting, port forwarding, and performance tuning
- **Plugin Development**: Java/Kotlin plugin development guides
- **Programming Languages**: Comprehensive language support information
- **Common Issues**: Real solutions to problems encountered by the community
- **Community Resources**: Tools, repositories, and helpful links

## Running Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Technology Stack

- **Framework**: [Next.js](https://nextjs.org)
- **Theme**: [Nextra](https://nextra.site)
- **Styling**: Built-in Nextra theme
